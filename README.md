Utopia Pharmacie is a comprehensive web-based ERP solution designed to digitalize and streamline daily pharmacy operations.
The project features a dual-interface architecture, providing a tailored experience for both healthcare staff and customers.

**Key Features:**

Staff Management: Analytical dashboard, real-time inventory tracking, and automated alerts for low stock or expiring products.

Sales & Invoicing: Advanced billing module supporting discounts and multiple payment statuses (Paid, Partial, or Unpaid).

Procurement: Supplier management and streamlined purchase order processing.

Customer Portal: Online shop for browsing the medication catalog and placing reservations.

**Tech Stack:**

Backend: ASP.NET Core 8.0 (Razor Pages), Entity Framework Core.

Database: Microsoft SQL Server.

Security: ASP.NET Core Identity (Roles: Admin, Pharmacist, Customer).

Frontend: Bootstrap 5, Vanilla JavaScript.
